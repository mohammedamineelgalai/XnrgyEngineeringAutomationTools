For example, if a property has a List of Values of A, B, and C, and the property has a value of D, then this determines whether or not that should be considered a policy violation.

PropertyDefinition.EnumeratedValue Represent a possible value that can occur in a PropertyDefinition.ValueList (or List Of Values). A List Of Values are the recommended (and sometimes enforced) values that the user should select from when entering a property value.
PropertyDefinitionDictionary This class is a collection (dictionary) of PropertyDefinition objects with a dual key. Each item in the collection has a "long" ID key and a "string" key. The long key is typically a database id, and can not be used to persist from session to session because migrations can cause the value to change. The "string" key is a system name that is unique across all PropertyDefinitions and can be persisted from session to session.
PropertyDefinitionIds This class contains constants for well-known System Names of PropertyDefinitions
